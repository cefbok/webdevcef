# Cef's Portfolio

A simple personal portfolio, still in development

## Tech/Framework used

Build with

- React
- Vite
- TailwindCSS

## Installation

Use the package manager [npm](https://docs.npmjs.com/cli/v8/commands/npm-install) to install node-modules.

```bash
npm install
npm run dev
```
